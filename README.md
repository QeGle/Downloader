### How to add library to your project

1. Add the JitPack repository to your build file
```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
2. Add the dependency 
```
dependencies {
	implementation 'com.github.QeGle:Downloader:0.1'
}
```
